Project

COCORE

COCORE is a Linux distribution for Programmable SoC (Zynq). In COCORE, a file system is based on Ubuntu 14.04 armhf, and COCORE includes Xillybus and ROS (Robot Operating System)

cReComp

creator for Reconfigurable Component. The cReComp is a code generator and framework for componentization of a single hardware or the multiple hardware.
